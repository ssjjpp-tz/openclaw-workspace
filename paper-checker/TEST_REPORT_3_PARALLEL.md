## 🔍 关键观察

### DOI 验证（瓶颈）

**观察：**
- 3 篇同时请求 CrossRef API
- 未遇到 API 限流
- 每篇约 78-85 秒完成

**结论：**
- CrossRef API 允许适度并发
- 3 篇并发是安全的
- 5 篇可能接近限制

### 数据库操作

**观察：**
- 3 线程同时写入数据库
- 无锁竞争错误
- SQLite 处理良好

**结论：**
- SQLite 支持适度并发写入
- 3 线程是安全的
- 建议使用 WAL 模式优化

### 文件 I/O

**观察：**
- 3 线程同时读取论文文件
- 同时写入压缩包
- 无 I/O 错误

**结论：**
- 文件系统支持并发读写
- 3 线程无压力
